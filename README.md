# Pulpino-for-dummies

## Requirements
* Xilinx Vivado 2018.3
* Git LFS and ssh
* Ubuntu 18.04
```bash
$ sudo apt install python-yaml
```
## Cloning
```bash
$ git clone git@github.com:hnkgroup/Pulpino-for-dummies.git
$ git submodule update --init
```
## Building
Open Vivado Project in ***fpga***
