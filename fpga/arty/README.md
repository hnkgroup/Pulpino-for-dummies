# Create Vivado ARTY project using Make

## Requirements
* Xilinx Vivado 2018.3
* Git LFS and ssh
* Ubuntu 18.04


to create vivado 2018.3 project in bash mode
```bash
$ make
```

to create vivado 2018.3 project using GUI
```bash
$ make gui
```

to clean vivado project (retain only original files) 
```bash
$ make clean
```
